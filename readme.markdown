

### or not
> or  相当于 ||
> not 相当于 !

> 常用的语法对比

JS          | python        | 描述
--          | --            | -- 
\|\|        | or            | 或
not         | !             | 非
[].length   | len([])       | 获取数组的长度（python中叫list）

> JS Array vs Python list

JS                      | python            | 描述
--                      | --                | -- 
[].length               | len([])           | 获取数组的长度（python中叫list）
[].push(n)              | [].append(n)      | 数组尾部追加数据
[].splice(n,0,1)        | [].insert(n,1)    | 指定位置追加数据
[].splice(n,1)          | [].pop(n)         | 制定位置删除数据
